# human motion TAE

train data downloaded from https://github.com/Mathux/ACTOR/blob/master/DATASETS.md

# Environment

- Windows
- CUDA 12.8
- RTX 4060 Ti (Ada Lovelace)

### pip venv

pip install -r requirements.txt
